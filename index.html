TFT Match History Analyzer

This is a personal project to track and analyze Teamfight Tactics (TFT) match history for my own account. The program retrieves match data using the Riot Games API, including champions, traits, items, placements, and outcomes.

The goal of the app is to provide actionable insights to improve my gameplay, such as: average placement, win rates, most effective champion and trait combinations, and item usage trends.

Important: This app only accesses my personal match data. It does not collect or share data from other players without their consent.

Project hosted on GitHub: https://github.com/YourUsername/TFT-Match-Analyzer
